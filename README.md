# Kwangdong Landing Page

광동제약 브랜드를 리뉴얼한 랜딩 페이지입니다.

## Overview
브랜드의 메인 비주얼과 핵심 콘텐츠를 중심으로 구성한 랜딩 페이지로,  
웹 퍼블리싱 기반의 레이아웃 구현과 사용자 흐름을 고려한 화면 구성에 중점을 두었습니다.

HTML, CSS, JavaScript를 활용해  
섹션별 구조를 설계하고 기본적인 인터랙션을 구현했습니다.

## Tech Stack
- HTML
- CSS
- JavaScript

## Main Features
- 메인 비주얼 중심의 랜딩 페이지 구현
- 섹션별 정보 구조 및 레이아웃 구성
- 기본 인터랙션 적용
